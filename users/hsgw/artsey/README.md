# ARTSEY keymap

This keymap is based on [artsey.io](https://artsey.io).

## Generate combo.h

1. Edit conbos.csv
2. run `python generate_combo.py`

## Special keycode

-   One-shot modifier
    -   AR_OSM_SFT
    -   AR_OSM_CTL
    -   AR_OSM_GUI
-   Clear all mods
    -   AR_PANIC

## Modifier

The right modifiers are configured as one-shot modifier.  
Please use the LShift, LControl, and LGUI keys for keymap with mod.
